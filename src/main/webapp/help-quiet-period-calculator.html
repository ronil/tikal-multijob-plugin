Enter a Groovy snipped that calculates a quiet period for a sub-job by it's index. The index is available as a variable named index. The resulting number is set as the quiet period in seconds.

Examples:

  • 0: no quiet period for all sub-jobs
  • index < 5 ? 0 : 2 * 60: start 4 jobs immediately, the other ones with a quiet time of 2 minutes.
